What is LandManageR?

LandmanageR is a web tool developed by Graham Montgomery & Ben Tonelli, PhD students at UCLA in Morgan Tingley’s global change ecology & conservation lab. It is intended to meet several needs:

  • creating a “one-stop shop” for data streams relevant for ecology & conservation in an era of rapid anthropogenic change & increasingly large amounts of data
  • localizing global data streams by making them more relevant to the places people live and work, especially land managers
  • making data streams more accessible
  • facilitating report creation
  • doing all of the above in a free, customizable, and sharable framework that we hope will be especially useful for managers with limited financial/IT resources.

Our vision & next steps

There’s a lot on the to-do list! We want to make this tool more useful by:

  1. incorporating more non-US data streams to make it more useful globally.
  2. fully automating data updates (no rerunning R scripts required)
  3. making the interface & workflow completely point-and-click (no coding required)
  4. and much more!
